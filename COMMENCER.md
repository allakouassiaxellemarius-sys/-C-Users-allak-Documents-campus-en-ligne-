## 🚀 Pour Démarrer (3 Étapes Simples)

### Étape 1️⃣: Installer Node.js

**Si Node.js n'est pas encore installé:**

1. Allez sur https://nodejs.org/
2. Téléchargez la version **LTS** (Long Term Support)
3. Installez-le et redémarrez votre ordinateur
4. Vérifiez: `node --version` et `npm --version`

### Étape 2️⃣: Installer les Dépendances

Ouvrez un terminal/PowerShell dans ce dossier et exécutez:

```bash
npm install
```

Attendez que tout s'installe (2-5 minutes selon votre connexion).

### Étape 3️⃣: Lancer le Site

```bash
npm run dev
```

Le site sera disponible sur: **http://localhost:5173**

## 🆘 Troubleshooting

### "npm: command not found"
- Node.js n'est pas installé
- Installez-le depuis https://nodejs.org/
- Redémarrez votre ordinateur

### "Port 5173 is in use"
- Le port est occupé
- Utilisez: `npm run dev -- --port 3001`

### "node_modules errors"
- Supprimez et réinstallez:
  ```bash
  rm -rf node_modules package-lock.json
  npm install
  ```
