## 🆘 PROBLÈMES COURANTS

### "npm: command not found"
→ Node.js n'est pas installé  
→ Installez depuis https://nodejs.org/  
→ Redémarrez votre ordinateur  

### "Port 5173 en utilisation"
→ Utilisez un autre port:  
```bash
npm run dev -- --port 3001
```

### "node_modules ne s'installe pas"
→ Nettoyez et réinstallez:  
```bash
rm -rf node_modules package-lock.json
npm install
```

### "Supabase URL/Key invalide"
→ Les clés sont optionnelles  
→ L'app fonctionne en mode démo  
→ Configurez seulement si vous avez les clés
